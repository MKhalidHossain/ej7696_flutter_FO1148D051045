# Code of Conduct

## Our standard

This project is committed to a respectful, professional, and harassment-free environment for everyone participating in the repository and related community spaces.

Examples of expected behavior:

- Give constructive, technically grounded feedback.
- Assume good intent while still being direct about problems and tradeoffs.
- Respect different experience levels, communication styles, and backgrounds.
- Focus reviews and discussions on the work, not the person.

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or sustained disruptive behavior
- Sharing private information without permission
- Repeatedly derailing technical discussions

## Scope

This code of conduct applies to repository discussions, issues, pull requests, code review, and other project-managed communication channels.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and pull requests that violate this code of conduct.

Repeated or severe violations may lead to temporary or permanent restriction from participating in the project.
